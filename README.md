# erlang-cli-hello-world

## Description
A POC erlang project

## Tech stack
- bash
- erlang 10.7.1

## Docker stack
- docker-cli
- erlang:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
